# Production Checklist

Per concept before export:

- [ ] Official MedVirtual logo used
- [ ] Image is clear (not cloudy/muddy)
- [ ] Face is not covered by text card
- [ ] Text readable on mobile
- [ ] No tiny bullet text
- [ ] No unverified claims on image ($/hr, HIPAA, savings %)
- [ ] No competitor asset used directly
- [ ] No patient data shown
- [ ] CTA is clear
- [ ] Correct aspect ratio (4:5 / 1:1 / 9:16)
- [ ] File name follows `MV_META_[ANGLE]_[STYLE]_[TEMPLATE]_V1`
